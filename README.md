# LA28 Olympic Live Booking Pilot

React + Vite app for autopilot-style LA28 session booking decisions.

## Local Development

```bash
npm install
npm run dev
```

## One-Click Vercel Deploy

1. Push this folder to a GitHub repo.
2. Replace `YOUR_GITHUB_REPO_URL` in the button link below with your repo URL (for example: `https://github.com/yourname/live-booking-pilot`).
3. Click the button and deploy.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=YOUR_GITHUB_REPO_URL)

## Production Build

```bash
npm run build
```
